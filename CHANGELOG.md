# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-04-08

### Added
- **Auto-Harness System**: Automatic knowledge accumulation (patterns, gotchas, retrospective)
- **UI/UX Design System**: Professional design rules enforcement with slash commands (`/ui-component`, `/ui-review`, `/ux-audit`)
- **Security Assessment**: OWASP Top 10 based vulnerability analysis (`/sec-audit`, `/sec-fix`)
- **Three-Layer Engineering**: Prompt, Context, Harness engineering principles
- **4-Phase Workflow**: Plan → Execute → Verify → Handoff
- **Anti-Pattern Prevention**: 5 common failure patterns with prevention strategies
- **Scenario Routing Table**: Work type based routing for optimal tool selection
- **Knowledge DB Structure**: Automated `docs/wiki/` management system

### Changed
- Completely restructured SKILL.md for better readability and coverage
- Enhanced README.md with detailed installation guide and feature overview
- Removed all personal information for public release
