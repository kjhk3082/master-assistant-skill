# Contributing to Master Assistant

We welcome contributions! Here's how to get started.

## How to Contribute

1. **Fork** this repository
2. **Create a branch**: `git checkout -b feature/your-feature`
3. **Make changes** to `SKILL.md` or `README.md`
4. **Test** your changes by installing the skill in Claude Code
5. **Commit**: `git commit -m 'feat: add your feature'`
6. **Push**: `git push origin feature/your-feature`
7. **Open a Pull Request**

## Contribution Guidelines

- Keep `SKILL.md` focused on Claude Code agent behavior
- All new features should include documentation in README.md
- Follow the existing structure and formatting
- Test changes in a real Claude Code session before submitting

## What We're Looking For

- New slash commands for specific workflows
- Better anti-pattern detection rules
- Industry-specific CLAUDE.md templates
- Translations (Korean/English/Japanese)
- Bug fixes and improvements

## Code of Conduct

Be respectful, constructive, and collaborative. We're all here to make Claude Code better.
